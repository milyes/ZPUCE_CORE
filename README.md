# ZCORE // Network Operations Center

**Tableau de bord de surveillance réseau en temps réel**  
Une interface cyberpunk moderne et interactive pour la supervision d'infrastructure réseau.

![ZCORE Dashboard](https://via.placeholder.com/800x400/050709/00d4ff?text=ZCORE+NETWORK+DASHBOARD)

---

## 🌐 Aperçu

ZCORE est un **dashboard NOC (Network Operations Center)** autonome, conçu pour simuler et visualiser en temps réel :

- Surveillance des hôtes
- Trafic réseau (IN/OUT)
- Latence et perte de paquets
- Flux réseau actifs
- Journal d’alertes
- Top ports utilisés

Entièrement développé en **HTML + CSS + JavaScript** (vanilla), sans dépendances externes.

---

## ✨ Fonctionnalités

- **Interface cyberpunk** avec effets glow, scanlines et animations fluides
- **Mise à jour en temps réel** toutes les 2 secondes
- Graphiques interactifs (trafic et latence)
- Simulation réaliste de trafic, latence et alertes
- Responsive (mobile & desktop)
- Horloge UTC + temps de fonctionnement
- Thème sombre cohérent avec accents cyan/vert
